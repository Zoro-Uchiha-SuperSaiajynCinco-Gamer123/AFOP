# AFOP
AFOP, que significa "Anotações de Fichas de Ordem Paranormal (O RPG)", é um projeto dedicado a armazenar dados de uma ficha de RPG, no molde de AOP (A Ordem Paranormal). Eu escolhi esse tema pois eu gosto de fazer e participar de RPGs, por isso esse projeto, além de ser algo pedagógico, também pode ser uma ferramenta útil para usar em campanhas.
